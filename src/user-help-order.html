Q1:
支持什么付款方式？
A.商城目前支持在线支付、货到付款两种付款方式
B.在线支付可选择的支付方式有：余额支付、支付宝支付、微信支付、银联支付
Q2:
已支付，订单状态还是“等待付款”怎么办？
如确定已支付，请联系在线客服为您查询或拨打客服热线：400-6799-859
Q3:
如何修改商品信息和收货信息？
A.未支付：直接取消订单重拍即可
B.已支付：如需更改收货人信息，请及时联系客服为您备注；但商品信息无法更改，如需更改请退款重拍
C.已发货：无法修改商品信息和收货信息
